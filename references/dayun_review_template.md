# 大运评价模板（Dayun Review Template · v7.5）

> Step 3a 写 `analysis.dayun_review[label].body` 时的标准模板。每段 10 年大运按"区间叙事 + 节点镶嵌"组织，**不再用单年六维取象**。

## 1. 整体结构（每段大运必备的 6 块）

```markdown
**{大运干支}（{start_year}–{end_year}） · 你 {age_start}–{age_end} 岁**

### A. 命理属性（≤ 4 句）
{大运五行 / 十神 / 与原局的关系 / 与上一段的接续}

### B. 时代镶嵌（v7.5 新增 · 必写）
{这 10 年覆盖的 era_windows + 大运 × era_window 的对齐情况}

### C. 三派交叉与盲派应事（≤ 4 句）
{扶抑/调候/格局在这段大运的整体倾向 + 触发的关键盲派事件}

### D. 4 维曲线整体倾向（精神 / 财富 / 名声 / 关系能量）
{每维 1-2 句，给方向 + 关键年份提示}

### E. 关键年份概览（≤ 5 个）
{这段大运里的 peak / dip / shift 列表，每个 1 句}

### F. 这 10 年的核心策略建议
{1-2 段，可操作的方向，禁止心灵鸡汤}

### G. 德性侧记（v8 新增 · v9 机械审计 · 条件触发 · ≤80 字）
{**v9 铁律**：当且仅当 output/virtue_motifs.json.motif_recurrence_map 显示这 10 年有 ≥1 母题
被激活时**必写**；没触发**完全不写**。
机械审计：[scripts/audit_virtue_recurrence_continuity.py](../scripts/audit_virtue_recurrence_continuity.py)
`_check_position2()` 扫每段 dayun_reviews 是否有 G 块（"### G." 或"德性侧记"标记），
缺一段触发即 exit 2。

按 [virtue_recurrence_protocol.md §位置 ②](virtue_recurrence_protocol.md) 规范：
- 第 1 次：相对中性命名
- 第 2 次：略带感情（"又来了"/"那个 X 岁的剧本以另一种皮重演"）
- 第 3+ 次：累积计数（"第 N 次了"）
- 第 5+ 次：转向"它不再像应期，更像你在变成什么"
- L 类母题：不计数，写"它从 X 岁就在你心里，到现在还在"
- 第 ≥2 次出现的母题侧记必须 ≥1 个自审句（"你以为 X，但 X 是不是其实是 Y？"）
- 必须 trace 回脚本提供的具体激活点（年龄 + 触发结构）
- 不下定论：不能给"所以你是 X 型的人"这类结论
- 回应感铁律：禁止"我看到你 X"告知感句式
- 禁用词表（详见 protocol §3.5）：塑造了你 / 一切有意义 / 时间会证明 / 等

视觉上用引用块（>）与 A-F 段分层，让读者看出这是与古法叙事并列的另一条线}
```

## 2. B 段「时代镶嵌」的写法（v7.5 重点）

按 `zeitgeist_protocol.md §4` 大运 × era_window 三种对齐情况分别处理：

### 情况 A · 大运 ≈ era_window（重合度 ≥ 70%）

**这段是命运共振区**，B 段写最长（≥ 200 字）。

```markdown
### B. 时代镶嵌
这段大运正好和「{era_window.label}」（{span}）高度重合——你的{age_start}–{age_end}
岁恰好是中国（或全球）{era_window.decade_label} 的一段。

具体的时代底色是：{era_window.keywords 展开 3-4 句具体描述}

这段时间的标志性细节，按你 yongshen={X} + class_prior={Y} 推，最可能体现为：
- {folkways 条目 1（high）}
- {folkways 条目 2（high）}  
- {folkways 条目 3（mid，加条件）}
- ...（4-6 条，跨 sub_layer）

这段大运 × 时代区间的核心张力是：{1-2 句把命运和时代联立讲}
```

### 情况 B · 大运跨越 2 个 era_window

```markdown
### B. 时代镶嵌
这段大运横跨两个时代区间：

**前 {N} 年（{year1}–{year2}）你处在「{era1.label}」的{尾声/中段}**——
{era1 简短背景，2-3 句}

**后 {M} 年（{year2+1}–{year3}）切换到「{era2.label}」**——
{era2 简短背景，2-3 句}

**时代切换感受**：你大概率在 {year2-1}–{year2+1} 这 2-3 年里感受到一种"换天"——
{大运 + 时代切换的双重影响，2-3 句}
```

### 情况 C · 1 个 era_window 跨越 2 个大运

```markdown
### B. 时代镶嵌
这段大运处在「{era_window.label}」（{span}）的 {前段/后段/中段}——
{era 整体背景已在 {上/下}一个大运段里详细写过，这里只补这一段大运的特定子节奏}

{1-2 句这段大运在 era 内部的特定位置感}
```

## 3. E 段「关键年份概览」的写法

E 段是给读者一个目录式的总览，每个 key_year 1 句话。完整解读在下方"关键年份评价"区。

```markdown
### E. 关键年份概览
- **{year1}（{age1}岁，{ganzhi}）· {kind}**：{1 句概括，与下方"关键年份"卡片的 headline 一致}
- **{year2}（{age2}岁，{ganzhi}）· {kind}**：{1 句概括}
- ...（最多 5 个，超过的合并）
```

## 4. F 段「核心策略建议」的写法

```markdown
### F. 这 10 年的核心策略建议
{1-2 段。结构：
 - "前半段（{age_start}–{midpoint}岁）的关键动作是 X"
 - "后半段的关键动作是 Y"
 - 可加一个具体的"避开 / 抓住"清单
 
 禁止：
 - "把握机会 / 顺其自然" 这种心灵鸡汤
 - "你会得到 X" 这种结果性预测（违反"后事粗"）
 - 命名身份 / 阶级化语言}
```

## 5. 完整示例（以 1985 男 / 辛巳大运 / 17-26 岁 / 2002-2011 为例）

```markdown
**辛巳大运（2002–2011） · 你 17–26 岁**

### A. 命理属性
辛金透干、巳火当令——金气接替了上一段的庚辰土金气，伤官 + 偏财格局开始浮现。
和原局比，辛巳是"主动型用神"——它让你能"用才华兑换资源"，但也带"火炼金"的
燥气，对睡眠和情绪有持续小考验。

### B. 时代镶嵌
这段大运横跨两个时代区间：

**前 6 年（2002–2008）你处在「入世红利 + 城市化加速」**——
WTO 红利期，房地产从启动到一轮狂飙，第一波互联网创业（2003-2005 BAT 雏形 +
2005 校内/Web2.0），SARS（2003 春），北京奥运成为这段时代的高潮。

**后 3 年（2009–2011）切换到「金融危机后 + 4 万亿」**——
全球金融危机后中国 4 万亿刺激下高铁+地产再起，智能手机普及（iPhone 4 在 2010
进入中国），微博 2009 末上线 → 2010 全民开始用。

**时代切换感受**：你大概率在 2008-2009 这两年感受到一种"换天"——
2008 是国家集体高光（奥运 + 神七），但年底全球金融危机；2009-2010 你的同辈
开始一边谈"四万亿"，一边换智能手机，一边在微博上发段子。这段切换刚好和
你大学到工作的过渡重合，是你"看到了一种父母不熟悉的新世界"的具体时间窗。

按你 yongshen=金 + class_prior=urban_state_or_educated 推，这段时间标志性细节
最可能是：
- **学英语 + 出国热**（2003-2008 高峰 / high）：高考英语 + 雅思托福 + 申请
- **第一波互联网产品**（2003-2008 / high）：QQ 空间、校内、人人、第一批博客
- **iPhone + 微博**（2009-2011 / high）：第一次有了"自媒体感"
- **房改窗口**（2007-2010 / mid · 因 21-25 岁通常还没买房）：但你大概率
  目睹了父母或亲戚的购房决策

### C. 三派交叉与盲派应事
扶抑派：用神金透出，整段偏吉，平均 +12。
调候派：燥气加重，2004-2006 一段需注意肝/眼/心血管疲劳。
格局派：伤官生财格成立，2004 / 2008 / 2011 三个年份触发关键应事。
盲派：2008（戊子）伤官见印 + 子午冲，2011（辛卯）禄被冲 + 偏财格起。

### D. 4 维曲线整体倾向
- **精神**：整段中等偏上（基线 +5），2004 短低谷，2008 高峰
- **财富**：明显上行（基线 +15），2008 + 2011 两个跳档点
- **名声**：温和上行（基线 +6），2008 起开始"被外部看见"
- **关系能量**：先弱后强（17-22 偏弱、23-26 偏强），2010 ± 1 是关系开始稳定的窗口

### E. 关键年份概览
- **2003（18 岁，癸未）· shift**：高考 + SARS，进入大学的关键节点
- **2008（23 岁，戊子）· peak**：财富/名声第一次明显跳档（多对应毕业 + 工作起步）
- **2010（25 岁，庚寅）· shift**：关系能量进入稳定窗口
- **2011（26 岁，辛卯）· peak**：偏财格起，第一次"主业之外的小钱"

### F. 这 10 年的核心策略建议
**前半段（17–22 岁）** 的关键动作是"在学习/工具上下功夫" —— 这段时间你的命局
最受益的是把"学英语 / 学计算机 / 学写作"这一类"积累型工具"做成你的长期资产，
而不是急着变现。和这段时代的"出国热 + 互联网产品潮"叠加，你最容易在这两个
方向之一找到自己的赛道。

**后半段（23–26 岁）** 的关键动作是"开始把工具变成产出" —— 工作起步后立刻
开始有意识地建"个人名片"（写作 / 项目 / 公众号萌芽 / 行业积累），不要把全部
能量都放在主业。命局的偏财气在 26 岁前后会有第一次明显的兑现窗口。

**避开**：把全部时间和注意力压在"考公 / 进体制"这一条线上 —— 这段大运的
能量结构 + 时代环境都不偏向这条路，硬走会感到"用错力"。
**抓住**：2008 ± 1 和 2011 ± 1 这两个时间窗的"被看见 + 小钱"机会，不要因为
"还在打基础"就错过。
```

## 8. 关键年份卡片中的"深层注脚"（位置 ③ · v8 新增）

E 段是目录式概览；每个 key_year 在主 body 里都有对应"关键年份评价"卡片。**当且仅当** `output/virtue_motifs.json.convergence_years[i].is_convergence == true` 时，在该年的卡片末尾、mangpai canonical_event 注脚之后，附加位置 ③ 深层注脚（≤120 字）。

详见 [virtue_recurrence_protocol.md §位置 ③](virtue_recurrence_protocol.md)。要点：

- 仅 `convergence_years` 标记为 true 的年份触发，非 convergence 年禁写
- 必须用**自审句式**（如"你那时是否……" / "那一年它是否……"开头），不下定论；v9.3 起严禁使用「陀氏 / 陀式 / 那一刀」等自封文学梗（被 `tone_blacklist.yaml` v9.3 banned_patterns 全位置封禁，命中即 exit 5）
- 必须 trace 引用脚本里的 convergence_motifs（≥2 条母题在该年同时激活）
- 不替代也不修改 mangpai canonical_event 文本，只是另一层声音
