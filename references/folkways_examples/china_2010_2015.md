## "范式输出"示例

### 移动互联网兴起 · 你 20–25 岁 · 2010–2015（跨大运辛卯-壬辰）

#### 时代底色

如果说 90 年代末是"换天的紧"，那 2010-2015 这一段是"换天的快"。智能手机从 iPhone4（2010）到全国普及只用了 3 年；微信从 2012 推出到 2015 已经吃掉了大部分人的日常通讯；O2O 创业潮席卷一二线，"BAT + 创新工场 + 双 11"成为新词频。同时金融危机后的 4 万亿余波 + 高铁全网 + 房价两轮上涨，让"做实业不如买房"开始成为城市中产的共识。这是中国"移动互联网 + 城市化加速 + 资产价格抬升"三股力量同时发力的一段，也是后来"内卷与躺平"叙事的物质基础——当时没人意识到，但 2013-2015 已经是某种"红利末段"。

这段时间里有几个标志性的细节，你大概率经历过：

- **智能手机 + 4G 上网**（2010-2014，high）：iPhone4/4s + 第一批小米 + 安卓百花齐放，2014 年 4G 商用开始让"手机看视频"变成日常
- **微信 / 朋友圈 / 公众号**（2012-2015 起飞期，high）：从最开始"加同事的尴尬"到 2015 年朋友圈成为社交主战场，公众号 2013-2014 是写作变现红利期
- **微博辉煌到衰退**（2010-2013 高峰，high）：那两年大事都在微博上爆，到 2014 已经明显变成"看明星 + 段子手 + 商家"的场所
- **创业 / O2O 大战**（2012-2015，high）：团购 → 打车 → 外卖前夜，"创业"成为城市年轻人的新职业选项之一
- **大学到工作的过渡（90 后特色）**（你 22 岁 = 2012）：找工作时的氛围还是"国企 / 外企 / 互联网"三角，2014-2015 互联网开始明显抢人
- **房价首轮大涨 + 二轮**（2013-2015，high）：城市里"早 5 年买房 vs 没买房"开始拉开第一道明显差距
- **韩剧 + 国产青春电影 + 偶像化**（2013-2015，high）：来自星星的你 / 致青春 / 我是歌手，集体审美开始向"韩潮"和"怀旧青春"分流

#### 命局给的几个具体节点

##### 2012 年（22 岁，壬辰，辛卯大运 → 壬辰大运换柱）· 比劫透干 + 大运换柱
**命理读出**：壬辰年大运换柱，比劫透干意味着"同辈圈子重新排列"。

**放在大背景里看**：
- **首选形态**（high，因 22 岁多对应大学毕业）：这一年你大概率经历了一次明显的"同辈圈子洗牌"——毕业 / 第一份工作 / 离开熟悉的城市 / 第一次在新环境里重组朋友圈。这是"大运换柱"在你的真实生活里的标志性体现
- **备选形态 1**（mid）：你在这一年第一次进入了一个"前辈/同辈水平比你高一截"的环境，逼着你重新校准自己的位置
- **备选形态 2**（mid）：恋爱/亲密关系上有过一次明显的换人或断裂

##### 2013 年（23 岁，癸巳，壬辰大运）· 食神泄秀 + 火透
**命理读出**：癸巳年食神泄秀，火透出来对应你的 yongshen=火。

**放在大背景里看**：
- **首选形态**（high）：这一年你做的某个东西**第一次被外部认可** —— 工作上某个项目被领导/客户表扬、写作发表、第一次因为"你做的某件事"被认识。这是 yongshen=火 在该年具体的显化方式
- **备选形态 1**（mid）：你这一年开始尝试"用一个新工具/新平台展示自己"——可能是开始写公众号 / 在知乎答题 / 做小生意试水
- **备选形态 2**（mid）：消费升级类的快乐——第一次去某个国家旅游、买了对你有意义的物件

##### 2015 年（25 岁，乙未，壬辰大运）· 偏财格起 + 财库见
**命理读出**：乙未年财库被引动，偏财气浮现。

**放在大背景里看**：
- **首选形态**（high，因 class_prior = urban_market_oriented）：这一年你开始有"主业之外的小笔收入" —— 副业 / 项目分成 / 第一次真正"不靠工资"挣到钱。这是偏财在 25 岁前后最常见的具体显化形态
- **备选形态 1**（mid）：买了第一笔像样的理财（余额宝 / 银行理财 / 早期股票），有过第一次"钱在动而不是躺死"的感觉
- **备选形态 2**（mid）：身边有人开始拉你"一起做点事"——合伙、投资、跟着做项目

#### 这段区间的累计影响

这是你完成"从校园到社会"切换的 5 年。和上一代（70/80 初）不同的是，你这一代的"社会"长得很不一样——岗位之外有"创业"作为一个真实选项，购物 / 信息 / 关系几乎都搬到了手机上。命局的偏财气 + 食神泄秀在这段时间起了关键作用——同样这段时代里，没有偏财气结构的同龄人多半内化的是"必须靠一份稳定工资慢慢攒"，你的内化是"机会其实满地都是，关键是去做点事"。

这种底层叙事会在你 30+ 岁时双向放大——它会让你比同辈更愿意尝试新东西（副业 / 跨行 / 小投资），但也容易让你低估"稳定本身的价值"。30 岁后如果遇到行业洗牌或经济周期向下，这种"机会满地"的内化感会被现实打折扣，那时你需要重新校准。

#### ⚠ 整段证伪点

如果你回忆 2010-2015 整段，以下任意一项明显对不上 → 阶级 prior 或 yongshen 推错：
- 你大学毕业前后（22 岁 ± 1）**完全没有**圈子洗牌 / 离开熟悉环境的体验
- 你在 23-25 岁**完全没有**任何"被外部认可 / 副业小钱 / 跟人合作做事"的体验
- 你这 5 年完全没用过微信 / 智能手机 / 网购（基本不可能，但如果是这样请告诉我）

告诉我哪一项对不上，我会重新校准。

## 注意事项（针对 LLM 写 2010 后内容时的特殊提醒）

### 距离当下越近，时代叙事越要克制

2010-2015 距离 2026 只有 11-16 年，**很多读者本人就经历过这段**。这种情况下：

- **不要给"全民共同记忆"刷长清单**——读者会觉得"你只是在复述大众媒体说过的话"
- **要给"和命主命理结构联动"的具体推论**——这是命理 + 时代联立才能产出的，单看时代或单看命理都给不出
- **避免"那时候大家都"句式**——多用"在你这种 X 命局结构 + Y 阶级 prior 的人身上，最可能体现为 Z"

### "前事细 / 后事粗"的边界点

2026 之前 = 后视镜（前事），可以给具体形态（首选 + 备选）。
2026 之后 = 未来（后事），只能给方向 + 大类。

如果读者出生在 2000+ 后，且你正在写他/她的 2025-2030：可以给"AI 时代职业大方向 / 技能组合大趋势"，但**不能**给"你会做什么具体的工作"。
