# 术语表（Glossary）

## 干支

- **天干**：甲乙丙丁戊己庚辛壬癸（10 个）
- **地支**：子丑寅卯辰巳午未申酉戌亥（12 个）
- **干支组合**：60 甲子，每年 / 月 / 日 / 时各对应一柱

## 五行

- 木（甲乙寅卯）/ 火（丙丁巳午）/ 土（戊己辰戌丑未）/ 金（庚辛申酉）/ 水（壬癸亥子）
- **生克**：木生火、火生土、土生金、金生水、水生木；木克土、土克水、水克火、火克金、金克木

## 四柱与日主

- **四柱**：年柱 / 月柱 / 日柱 / 时柱，各含天干 + 地支
- **日主 / 日元**：日柱的天干，代表「自我」

## 十神

以日主为基准，对其他天干 / 地支主气定义十种关系：

| 十神 | 关系 | 大致象征 |
|---|---|---|
| 比肩 | 同我同性 | 同党、朋友、自我认同 |
| 劫财 | 同我异性 | 竞争者、消耗 |
| 食神 | 我生同性 | 创造、表达、温和产出 |
| 伤官 | 我生异性 | 才华、爆发、出圈 |
| 正财 | 我克异性 | 稳定收入、配偶（男） |
| 偏财 | 我克同性 | 偏门财、机会、父亲 |
| 正官 | 克我异性 | 正职、社会身份、丈夫（女） |
| 七杀 / 偏官 | 克我同性 | 压力、权力、敌手 |
| 正印 | 生我异性 | 母亲、文凭、平台 |
| 偏印 / 枭神 | 生我同性 | 偏门学问、副业平台 |

## 用神 / 喜神 / 忌神 / 通关

- **用神**：能让命局趋于平衡的核心五行
- **喜神**：生扶用神的五行
- **忌神**：克泄用神 / 加剧失衡的五行
- **通关用神**：当原局两组五行强烈相克时，居中调和的第三方五行（如金木交战需水通关）

## 调候

- 根据出生月份的寒暖燥湿来调节命局：冬生需火暖、夏生需水润、春生需金疏、秋生需木通

## 格局

- 命局的整体结构模式，常见：正官格、七杀格、正财格、偏财格、食神格、伤官格、正印格、偏印格、建禄格、阳刃格
- 「成格」= 格局成立且大运配合；「破格」= 关键字被冲克

## 大运 / 流年

- **大运**：从月柱起排，每 10 年一柱（阳男阴女顺、阴男阳女逆）
- **流年**：每年的干支
- **小运 / 月运**：本 skill 不细化到月度

## 刑冲合害穿

- **冲**：相对地支两两相冲（子午、丑未、寅申、卯酉、辰戌、巳亥），冲则动、变
- **刑**：寅巳申三刑、丑戌未三刑、子卯相刑、辰午酉亥自刑
- **合**：六合（子丑、寅亥、卯戌、辰酉、巳申、午未）；三合局（申子辰水、亥卯未木、寅午戌火、巳酉丑金）；三会方（亥子丑北方水、寅卯辰东方木、巳午未南方火、申酉戌西方金）
- **害 / 穿**：六穿（子未、丑午、寅巳、卯辰、申亥、酉戌），破坏和气
- **伏吟**：流年柱与原局某柱完全相同（如年柱壬子，流年又遇壬子），主旧事重演 / 内省
- **反吟**：流年柱与原局某柱天克地冲（如原局壬子，流年丙午），主剧烈变化

## 库

- 辰戌丑未为四库：辰为水库、戌为火库、丑为金库、未为木库
- 库被冲开则其本气元素显化：如戌冲开火气、丑冲开金气

## 双层曲线（本 skill 专用）

- **实线**：当年值 = α·原局基线 + β·大运修正 + γ·流年修正
- **虚线**：累积值 = 历年实线值的指数衰减加权（沉淀感）
- 维度衰减系数：精神 0.85 / 财富 0.92 / 名声 0.96
