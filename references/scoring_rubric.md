# 三维度评分细则（加减分表）

每条规则带章节号（§A.1, §B.2 …），脚本输出的「加分项 / 减分项」明细必须引用对应章节号，便于追溯。

## A. 精神舒畅度 Spirit

### A.1 印星出现（庇护、自我接纳）

- 流年正印：+8
- 流年偏印（枭神）：+4（偏印有支持也有耗食的副作用）
- 大运正印：+12（区间常态）

### A.2 比劫出现（同党、自我认同）

- 流年比肩：+5
- 流年劫财：+3（劫财带竞争性）
- 大运比肩 / 劫财：+8（合身但易耗财）

### A.3 七杀攻身

- 流年七杀且无印化：-12
- 流年七杀且有印化解：-2
- 大运七杀且日主弱：-15

### A.4 食伤外泄过度（日主弱）

- 日主弱时流年食神：-6
- 日主弱时流年伤官：-10
- 日主强时流年食伤：+8（顺势宣泄）

### A.5 通关五行到位

- 流年补足通关用神：+10
- 大运为通关用神：+15

### A.6 关键负面互动

- 流年与原局相穿：-8
- 流年伏吟年柱：先 -5（上半年抑）后 +5（下半年扬），净 0 但波动大
- 流年三合忌神局：-10

## B. 财富 Wealth

### B.1 财星出现且身能任财

- 日主强时流年正财：+10
- 日主强时流年偏财：+12
- 日主弱时流年财星：-5（财多身弱反破财）

### B.2 食伤生财链路成立

- 流年食神 + 原局有财：+8
- 流年伤官 + 原局有财（无官星被克）：+12

### B.3 财库

- 流年冲开财库（库 = 辰戌丑未带本气财）：+15（一次性）
- 大运冲财库：+10（区间常态）
- 流年财库被冲开后又遇刑：-8（财来财去）

### B.4 比劫夺财

- 日主强 + 流年比肩夺财：-8
- 日主强 + 流年劫财夺财：-12

### B.5 大运成财格

- 用神为财、大运财星临旺：+20（区间常态）
- 忌神为财、大运财星：-15（区间常态）

### B.6 关键负面

- 流年破耗财星（金木战 / 水火冲）：-8
- 流年与财星相穿：-5

## C. 名声 / 影响力 Fame

### C.1 官星显化（社会身份）

- 流年正官（日主强）：+10
- 流年正官（日主弱）：-3
- 流年七杀化印：+8

### C.2 印星显化（平台、被认可）

- 流年正印：+8
- 大运正印：+12（区间常态）

### C.3 食伤外显（产出可见度）

- 日主强 + 流年食神：+10（声誉沉淀）
- 日主强 + 流年伤官：+15（爆款 / 出圈）
- 日主弱 + 流年食伤：-3

### C.4 官印相生

- 流年同时见官印：+15
- 大运官印相生：+18（区间常态）

### C.5 关键负面

- 流年伤官见官（无财通关）：-15
- 流年冲官星：-8
- 大运冲克日主：-10

## D. 全维度共享规则

### D.1 用神大运起讫

- 用神大运起年：精神 +12 / 财富 +10 / 名声 +10
- 用神大运讫年（最后一年）：精神 -5 / 财富 -3 / 名声 -3
- 忌神大运起年：精神 -15 / 财富 -12 / 名声 -10

### D.2 大运换运过渡

- 大运首尾两年权重 × 0.7（过渡期，势能未完全显化）

### D.3 关键互动加权倍率

| 互动 | 加权倍率 |
|---|---|
| 三合 / 三会成局且为用神 | × 1.3 |
| 反吟年（与原局柱天克地冲） | × 1.5 |
| 伏吟年 | × 1.2 |
| 流年大运同字（双重显化） | × 1.4 |

### D.4 派别分歧惩罚

- 当三派给分极差 > 20 时：最终置信度标 `low`，且年度值的绝对偏离（相对基线 50）× 0.7（保守化处理）

## E. 事后归因禁令

- 用户提供的「过去某年发生过 X」**不得**用于调整该年评分
- 仅可作为 `calibrate.py` 的个人化校准输入（影响个人副本，不污染主模型）
- 详见 `fairness_protocol.md` § 3
