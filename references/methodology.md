# 双层评分法学理与权重模型

## 1. 三维度的命理对应

| 维度 | 实线（当年体感） | 虚线（累积沉淀） | 主要十神锚点 |
|---|---|---|---|
| **精神舒畅度** | 流年印星 + 比劫 + 通关五行 - 七杀攻身 - 食伤被克 | 自我成熟度的滚动累加（衰减加权） | 印（庇护）/ 比（同党）/ 食（自我表达） |
| **财富** | 流年财星活跃 + 食伤生财链路 + 身能任财 | 净资产存量（衰减更慢） | 正财 / 偏财 / 食伤生财 |
| **名声 / 影响力** | 当年印星显化 + 官星 + 食伤外显 | 行业地位 / 影响力沉淀（衰减最慢） | 官（社会身份）/ 印（平台认可）/ 食伤（产出可见度） |

## 2. 三层独立打分（避免循环污染）

每年评分由三个独立计算结果加权融合：

- **L0 原局基线**：日主强弱、格局成败、用神 / 忌神 / 调候 / 通关需求 → 一个**恒定基线**（人一生不变）
- **L1 大运修正**：当年所在大运（10 年一柱）相对于原局是**生扶 / 抑制 / 通关 / 破局**
- **L2 流年修正**：当年干支与原局 + 大运的**刑冲合害穿**互动

**禁止反向回写**：流年得分不能反过来调整大运得分，大运得分不能反过来调整原局基线。三层各自独立，再合成。

## 3. 权重融合公式

```
year_value = α · L0_baseline + β · L1_dayun + γ · L2_liunian
```

- 默认 α = 0.30, β = 0.40, γ = 0.30
- 个人化校准（`calibrate.py`）可微调，约束 α + β + γ = 1

## 4. 累积线（虚线）的衰减加权

虚线代表「沉淀」，遵循指数衰减加权累加：

```
cumulative[t] = Σ_{k=0..t} λ^(t-k) · year_value[k] · normalization
```

- 默认衰减系数 λ：精神 0.85（情绪记忆衰减快）/ 财富 0.92（资产留存中等）/ 名声 0.96（声誉沉淀最持久）
- normalization 把累积值映射到 [0, 100]

## 5. 三派交叉打分（核心准确性机制）

每年的 L0/L1/L2 都由 **三派独立计算**，再融合：

| 派别 | 关注点 | 给分逻辑示例 |
|---|---|---|
| **扶抑派** | 日主强弱 × 用神到位度 | 日主弱用印比，流年印星 +15 |
| **调候派** | 寒暖燥湿 × 流年五行调和 | 冬生水旺缺火，流年丙丁 +12 |
| **格局派** | 原局格局成败 × 大运是否破/成 | 伤官生财格遇大运财星临库 +18 |

**派别分歧检测**：

- 三派给分极差 ≤ 10：高置信度，最终值取加权平均
- 三派给分极差 11–20：中置信度，最终值取中位数
- 三派给分极差 > 20：**低置信度**，标记 `divergence: high`，图上灰阶 + 文字提示「派别分歧」

## 6. 关键互动事件加权

以下事件触发额外加分 / 减分（详见 `scoring_rubric.md` § 加权事件表）：

| 事件 | 典型影响 |
|---|---|
| 伏吟（流年柱 = 原局某柱） | 精神 V 字（先抑后扬）；财富 / 名声轻微波动 |
| 反吟（流年柱冲原局某柱） | 三维度都波动放大；事件密度高 |
| 三合 / 三会成局 | 主导五行方向受益的维度 + |
| 相穿 / 相害 | 精神 - ；其他维度看具体五行 |
| 用神大运起 / 讫 | 起：所有维度 +；讫：所有维度 - |
| 忌神大运起 | 所有维度 -；财富受冲击最大 |

## 7. 用神 / 通关 / 喜神 / 忌神 速查

- **用神**：能让命局趋于平衡的核心五行（扶弱 / 抑强 / 调候 / 通关）
- **喜神**：能生扶用神的五行
- **忌神**：克泄用神 / 加剧失衡的五行
- **通关用神**：当原局两组五行强烈相克时，居中调和的第三方五行

完整术语见 `glossary.md`。

## 8. 大运起运方向（性别相关）

- **阳男阴女**：顺排（月柱往后数大运）
- **阴男阳女**：逆排（月柱往前数大运）

这是性别在评分中**唯一**的影响。本命基线、用神判定、三派打分逻辑均与性别无关（公正性要求）。

## 9. 评分标度

- 每条线值域：[0, 100]
- 基线分：50
- 单年极端值不超过 [10, 95]，避免视觉失真
- 累积线由衰减加权计算后再标准化到 [0, 100]
