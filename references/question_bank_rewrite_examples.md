## 改写四原则（每条改写示范前自检）

1. **单概念 / 单维度**：每个 option 只承载一个属性（不要 "且 / 同时 / 又"）
2. **可观察具象**：用具体行为 / 时间窗口 / 频率，不用 "经常 / 优秀 / 中等" 这类
   靠用户主观映射的词
3. **互斥穷举**：4 个 option 在语义上互斥且尽量穷举常见情况
4. **零中性引导**：prompt 不暗示哪个 option "更好 / 更对"，options 顺序与
   likelihood 强弱无关（避免锚定）

## 范例 1 · `D4_Q1_cold_heat` · 寒热倾向

### 现状

> 你的整体寒热倾向是？
> A) 怕冷、手脚常凉、爱穿厚
> B) 怕热、爱出汗、爱冷饮
> C) 上热下寒（脸红脚冷 / 口干腿凉）
> D) 上寒下热（咽痒脚心热）/ 寒热不定

### 审计 finding

- `A3_multi_concept`: D 选项 "上寒下热" + "寒热不定" 是两个独立概念；C 选项把
  "脸红脚冷" 与 "口干腿凉" 并列括号里，对自检不熟的用户难判断
- 原 prompt "整体" 没给时间窗口

### 改写

> **过去 1 年里，下面哪种描述最贴近你的体感？**
> A) **常感冷**：手脚冰、比同事 / 家人多穿一件，喝热的更舒服
> B) **常感热**：手心脚心烫、爱出汗、喝冷的更痛快
> C) **上下不一致**：脸 / 头容易热（红、口干），同时手脚 / 小腹偏凉
> D) **没有稳定倾向**：随天气 / 经期 / 情绪变，自己说不上偏寒还是偏热

**改动要点**：
- 加 "过去 1 年" 时间窗口（修 A4_temporal_ambiguous）
- D 拆"上寒下热"和"寒热不定"，只保留"无稳定倾向"（修 A3_multi_concept）
- 选项加体感行为锚点（"多穿一件 / 爱冷饮"）减主观投射

## 范例 2 · `D4_Q2_sleep` · 睡眠

### 现状

> 你过去 3 年的整体睡眠状况是？
> A) 入睡快、深睡足、醒后清爽
> B) 多梦 / 浅眠 / 易醒，醒后疲乏
> C) 入睡难，但睡着后较深
> D) 睡眠时长足，但晨起仍困倦 / 沉重

### 审计 finding

- A 选项混 3 个概念（入睡快 + 深睡 + 醒后清爽），B 选项混 4 个；很多用户
  "入睡快但浅眠"，无法对号入座
- "整体" + "过去 3 年" 跨度太大，覆盖期内可能换工作 / 生子 / 病程

### 改写

> **过去 3 个月里，下面哪条最像你大多数夜晚的睡眠？**
> A) **躺下 30 分钟内能睡着，半夜不醒，闹钟响时已自然清醒**
> B) **躺下 30 分钟内能睡着，但半夜常醒 ≥ 2 次 / 多梦**
> C) **躺下后要 30 分钟以上才睡着，但睡着后基本到天亮**
> D) **能睡满 7 小时以上，闹钟响仍困到要按 2 次贪睡**

**改动要点**：
- 时间窗口 3 年 → 3 个月（修 A4_temporal_ambiguous）
- 每个 option 用 "入睡延时 / 中途醒次数 / 晨起状态" 三轴中的**一轴主导**
  （A=三轴都好,B=入睡好但中途醒,C=入睡难,D=入睡好但晨起困）→ 互斥
- 用具体阈值（30 分钟、≥2 次、7 小时）替代 "快 / 难 / 足"

## 范例 3 · `D5_Q1_default_strategy` · 压力下默认反应

### 现状

> 遇到大压力时，你最本能的反应是？
> A) 主动迎击，靠自己硬扛
> B) 借外力 / 找资源 / 找支持系统
> C) 顺势而为 / 等局势变
> D) 切换轨道 / 换个赛道重来

### 审计 finding

- 所有 option 都是抽象策略词，没有"最近一次大压力"这种**经验锚点**
- A8 风险（隐性）："主动 / 借力 / 顺势 / 切换" 在 social desirability 上接近
  "勇敢 / 聪明 / 智慧 / 灵活"，每个用户都能脑补出"自己是这样"

### 改写（用最近经验事件锚定）

> **回忆你最近一次明显的"压力事件"（工作 / 学业 / 关系），事后回看，
> 你当时其实做的最像下面哪种？**
> A) **顶住自己扛**：基本没找别人帮忙，靠加班 / 重做硬过去
> B) **拉外援**：第一时间找朋友 / 同事 / 家人 / 顾问商量或借资源
> C) **观望等待**：暂停推进，等情况自己变化（对方先动 / 形势变）
> D) **跳船 / 换赛道**：放弃当前路径，换工作 / 换关系 / 换城市

**改动要点**：
- 用 "最近一次" 锚定具体事件（避免"我大多数时候是 A"的自我形象偏差）
- 用 "事后回看，做的最像" 取代 "本能反应"（修 A8_leading 的 social desirability —
  "本能" 暗示褒义；"做的"是行为事实）
- 每个 option 加典型行为示例

## 范例 4 · `D2_Q1_partner_attraction_type` · 反复被吸引的画像

### 现状

> 你反复被吸引的对象，最常见的画像是？
> A) 强势主导型，你常跟随对方节奏
> B) 资源型 / 给予物质型，你欣赏对方供给能力
> C) 同辈对等型，势均力敌互不主导
> D) 你欣赏其才华、想推他 / 她"输出"的人

### 审计 finding

- D 选项 "你欣赏其才华" + "想推他 / 她输出" 是两件事
- B 选项 "资源 / 给予物质 / 供给" 三个近义词堆叠，用户读起来像被推销
- "反复" 没给计数（2 段算反复？5 段才算？）

### 改写

> **回想你过去喜欢过 / 在一起过的对象（包括暗恋对象，至少 2 个），
> 他们之间最常见的共同点是？如果完全没有共同点，选 E。**
> A) **多数比你强势 / 主导**：你愿意跟随对方节奏、决定多由对方拍板
> B) **多数物质条件比你强**：收入、资源或社会位置稳定高于你
> C) **多数和你旗鼓相当**：势均力敌、决策五五开
> D) **多数被你"看见"**：你觉得他们有才但被低估，乐于支持他们成长
> E) **完全没共同点 / 差异大**

**改动要点**：
- "至少 2 个" 给反复的计数定义
- D 拆成 "看见 + 支持成长"，去掉 "推他输出" 的工具感措辞
- 加 E 兜底（修 A7_option_overlap & 互斥性）

## 范例 5 · `D1_Q1_birth_economic_condition` · 出生时家境

### 现状

> 你出生时（前后 2 年内）家里的经济状况大致是？
> A) 富裕、宽绰，物质从未匮乏
> B) 中等偏上，紧但不缺
> C) 紧巴巴，常为钱发愁
> D) 困窘，缺过基本物资

### 审计 finding

- "中等" 是典型 A2_subjective_judgment（每个用户对"中等"的定位差异极大）
- "宽绰 / 紧 / 紧巴巴 / 困窘" 是连续光谱，但选项是断点；
  人在做绝对判断时易偏向社会期望（中产偏好选 B/C）

### 改写（改成相对参照系 + 具象事件）

> **你出生那两年，家里在 "买玩具 / 添新衣 / 全家下馆子" 这类**非必需开销**上的
> 频率，对照你身边邻居 / 同班同学家是？**
> A) **明显多于多数同学**：经常买玩具、衣服多到穿不完、节假日下馆子
> B) **稍多 / 与多数同学差不多**：玩具 / 新衣有，但家长会算着买
> C) **稍少 / 节俭明显**：玩具基本是亲戚送的，新衣多在过年才买
> D) **明显少**：玩具靠自制 / 借，新衣常是哥姐穿过的，下馆子是稀罕事

**改动要点**：
- 把抽象 "经济状况" 改成**可观察的具体消费行为**
- 加相对参照系（vs 同学），去掉 A2 主观词 "中等 / 宽绰"
- 用频率描述（"经常 / 算着 / 基本是 / 稀罕事"）代替绝对量级
